Rodando projeto local

Configuração inicial

git clone git@github.com:daniellbastos/work-at-olist.git
cd work-at-olist/
pip install -r requirements-local.txt
cd work-at-olist/
./manage.py migrate --settings=workatolist.settings.local
Importando dados
./manage.py importcategories walmart fixtures/categories.csv --settings=workatolist.settings.local
Startando o servidor
./manage.py runserver --settings=workatolist.settings.local

API

  • Lista dos channels: /api/channel/
  • Detalhes do channel: api/channel/PK/
  • Detalhes da category: /api/category/PK/

Sugestões

Para facilitar o isolamento dos diferentes ambientes em que o projeto pode rodar, recomendaria a criação de um pacote dos arquivos settings. Nesse caso teríamos os seguintes arquivos:

  • settings/base.py
  • settings/local.py
  • settings/heroku.py
